    Diagramas (Mermaid)¶

    En esta página encontrarás diagramas que muestran el flujo de uso de la app, la arquitectura básica y el flujo de pantallas. Usa la barra de búsqueda para localizar cada diagrama por su título.

    Flujo de uso de la aplicación¶

    flowchart TD
  Start([Inicio]) --> Login{Usuario logueado?}
  Login -- Sí --> Dashboard[Dashboard / Seleccionar rutina]
  Login -- No --> Auth[Login / Registro]
  Dashboard --> OpenRoutine[Abrir rutina]
  OpenRoutine --> StartSession[Iniciar sesión]
  StartSession --> Record[Registrar sets y reps]
  Record --> Save[Guardar sesión]
  Save --> Stats[Ver estadísticas]

    Flujo de pantallas (pantallas clave)¶

    sequenceDiagram
  participant U as Usuario
  participant A as App
  participant S as Servidor

  U->>A: Abrir app
  A->>S: Solicitar estado sesión
  S-->>A: Token / no token
  alt Tiene token
    A->>A: Mostrar Dashboard
    U->>A: Seleccionar rutina
    A->>S: Cargar datos rutina
    S-->>A: Datos rutina
    A->>U: Mostrar rutina
  else No token
    A->>U: Mostrar pantalla Login
  end
